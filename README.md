This repository contains a structured GRC portfolio built to demonstrate practical, hands-on knowledge of the four frameworks most relevant to the Irish and EU regulatory market — ISO 27001:2022, GDPR, NIS2, and DORA.

Each project produces a real, employer-facing deliverable: a document or spreadsheet that could be used in a professional GRC engagement. All projects use fictional companies and anonymised data.

## Projects

Project 1 — Multi-Framework Controls Mapping

Maps 10 core information security control domains across ISO 27001:2022, GDPR, NIS2 Article 21, and DORA. Identifies overlapping requirements, framework-unique obligations, and provides a practical recommendation for how an Irish organisation can implement once and satisfy all four frameworks simultaneously.

| Deliverable | Description |
|---|---|
| `GRC_Controls_Mapping.xlsx` | 10-domain controls mapping spreadsheet with overlap analysis |
| `Project7_Executive_Summary.docx` | 1-page board-level summary of findings and recommendations |

**Frameworks:** ISO 27001:2022 · GDPR · NIS2 · DORA  
**Key skills demonstrated:** Multi-framework mapping · Overlap analysis · Regulatory interpretation · Executive communication

Project 2 — ISO 27001 ISMS Documentation Pack *(in progress)*

Full ISMS documentation set for a fictional 50-person Irish SaaS company — including asset inventory, risk register, Statement of Applicability, and three security policies aligned to ISO 27001:2022 Annex A.

**Frameworks:** ISO 27001:2022

Project 3 — NIS2 Gap Assessment *(coming soon)*

Gap assessment and remediation roadmap for a fictional Irish healthcare technology company against all 10 NIS2 Article 21 mandatory security measures. Includes maturity scoring, prioritised remediation roadmap, and board-level briefing document.

**Frameworks:** NIS2 · ISO 27001:2022  
**Key skills demonstrated:** Gap analysis · Maturity modelling · Remediation planning · Board communication

Project 4 — Vendor / Third-Party Risk Assessment *(coming soon)*

Third-party risk assessment framework including vendor tiering model, 30-question security questionnaire, risk scoring matrix, and findings report — aligned to ISO 27001 Annex A.5.19–5.22 and DORA Articles 28–30.

## About This Portfolio

This portfolio was built as part of a structured self-development programme alongside an MSc in Cybersecurity from Dublin Business School (graduated January 2026). The goal was to move beyond theoretical knowledge and produce tangible, professional-quality GRC deliverables that demonstrate real analytical and documentation capability.

Every project is scoped to reflect the current Irish and EU regulatory environment — particularly the convergence of NIS2, DORA, and GDPR obligations that Irish organisations are navigating in 2025/26.
